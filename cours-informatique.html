Accompagnement en Informatique

Classes Préparatoires aux Grandes Ecoles (CPGE)

Maîtriser le code et l'algorithmique

  • Programmes Couverts :
    • Outil Informatique de base : Python, SQL, Graphes, Simulation numérique.
    • Option Informatique (MPSI/MP/MPI) : OCaml, C, Structures de données complexes, Logique.
  • Objectifs de l'Accompagnement : Acquérir une syntaxe irréprochable, comprendre la complexité algorithmique et réussir les épreuves écrites (X-ENS, Mines, Centrale) et orales.
  • Approche Pédagogique : Alternance entre théorie fondamentale et "Live Coding" pour développer les bons réflexes de programmation.

Filières Universitaires et Écoles d'Ingénieurs

  • Niveaux Universitaires :
    • Licence Info : L1 à L3 (Algorithmique, Web, Système, Réseaux).
    • Master : Data Science, Intelligence Artificielle, Développement Logiciel.
    • Projets : Aide au développement de projets de fin d'année.
  • Écoles d'Ingénieurs : Soutien pour les cursus ingénieur (Tronc commun info, spécialisation Data/IA). Préparation aux entretiens techniques (Coding Interviews).
  • Technologies & Concepts : Programmation Orientée Objet (Java/C++), Web (JS/React), Base de données (SQL/NoSQL), Machine Learning (PyTorch/Scikit).
